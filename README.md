# 📚 Online Library System

##  Project Description
This is a React + Vite based Online Library application where users can view and add books.
The project uses Redux Toolkit for state management and React Router for navigation.

---

##  Tech Stack
- React
- Vite
- Redux Toolkit
- React Router DOM

---

##  Features
- View list of books
- Add new book
- State management using Redux
- Client-side routing
